# Spotify Clone

A frontend clone of Spotify web player.

## Functionality

The app uses the Spotify API wich allows you to log in with your spotify account, shows your playlists and also list the songs you have recently listen to.


## Technologies used:
- React
- React Context API
- Spotify API
- Material UI
